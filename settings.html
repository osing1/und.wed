Konfigurasi API

Hubungkan ke npoint.io (backend gratis)

📋 Cara Setup npoint.io:
  1. Buka https://www.npoint.io/
  2. Daftar akun gratis (bisa pakai Google)
  3. Klik "Create new bin"
  4. Copy isi file data.json dari repo ini, paste ke sana
  5. Klik Save, copy URL bin (format: https://api.npoint.io/xxxxx)
  6. Buat 1 bin lagi untuk ucapan (kosongkan saja, isi [])
  7. Paste kedua URL di bawah ini
